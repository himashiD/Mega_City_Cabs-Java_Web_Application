# 🚖 Mega City Cabs – Java Web Application
## Project Overview

Mega City Cabs is a web-based cab reservation system developed to automate the manual booking process used by the Mega City Cab service in Colombo. The system allows customers to book vehicles online, drivers to manage assigned rides, and administrators to manage the overall system operations.
The application is developed using Java web technologies and provides an efficient platform for managing cab reservations, customer information, driver assignments, and billing operations.

## System Actors

### 👥Customer

Customers use the system to book cab services and manage their ride information.
- Customer Features
- Register and login to the system
- Book a cab for travel
- View booking history
- Check booking status
- Update personal profile information

### 🚗Driver

Drivers manage ride requests assigned by the system and update trip details.
- Driver Features
- Login to the system
- View assigned bookings
- Accept or reject ride requests
- Update trip status
- View trip history

### 🧑‍💼Admin

The administrator manages the entire system including users, drivers, and bookings.
- Admin Features
- Manage customer accounts
- Manage driver accounts
- View and manage all bookings
- Assign drivers to bookings
- Calculate bills and fares
- Generate system reports

## 🛠️ Technologies Used

### 💻 Frontend
- **HTML** – Structure of the web pages  
- **CSS** – Styling and layout design  
- **JavaScript** – Client-side interactivity  

### ⚙️ Backend
- **Java** – Core programming language for application logic  
- **Servlets** – Handle server-side requests and responses  
- **JSP (Java Server Pages)** – Dynamic web page generation  

### 🗄️ Database
- **MySQL** – Relational database used to store user, driver, and booking data  

### 🧰 Development Tools
- **Apache Tomcat** – Web server and servlet container  
- **Eclipse** – Integrated Development Environment (IDE)  
- **MySQL Workbench** – Database design and management tool  
- **Git & GitHub** – Version control and project repository management
